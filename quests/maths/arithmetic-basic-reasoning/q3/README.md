# Palindrome Number

**Difficulty:** Easy

## Problem
Given an integer `x`, return `true` if `x` is a palindrome, and `false` otherwise.

A palindrome is a number that reads the same forward and backward.

## Examples

### Example 1
**Input:**  
`x = 121`  
**Output:**  
`true`  
**Explanation:**  
121 reads the same from left to right and right to left.

### Example 2
**Input:**  
`x = -121`  
**Output:**  
`false`  
**Explanation:**  
From left to right: `-121`  
From right to left: `121-` → not the same.

### Example 3
**Input:**  
`x = 10`  
**Output:**  
`false`  
**Explanation:**  
From right to left: `01` → not the same.

## Constraints
- `-2^31 <= x <= 2^31 - 1`
